# 拆解对象：《鸣潮》今汐 全技能组

### 一、基础信息

| 维度     | 内容                                                         |
| :------- | :----------------------------------------------------------- |
| 角色定位 | 主C                                                          |
| 核心机制 | 普攻/技能叠层→特殊技能触发强化普攻模组→最终技能收尾→大招爆发。输出节奏为“叠层→启动→爆发”三阶段。 |
| 特殊机制 | **韶光值**：自身攻击每3秒周期可增加1点韶光值，队友协同攻击增加2点韶光值。韶光值影响最终技能伤害倍率。反推为一个独立的 `FGameplayAttributeData` 属性。 |

### 二、普攻连段

| 段数  | 表现/观察              | 伤害类型       | 反推实现                                                     |
| :---- | :--------------------- | :------------- | :----------------------------------------------------------- |
| 第1段 | 挥刀平砍               | 扇形范围       | 武器胶囊体在动画第X帧开启，做扇形Overlap检测                 |
| 第2段 | 刀脱手丢至前方旋转     | 圆形范围       | 刀Actor在动画第X帧Spawn，携带旋转动画+圆形碰撞体，持续Y帧后自毁 |
| 第3段 | 召唤角于前方旋转       | 圆形范围带牵引 | 角Actor在动画第X帧Spawn，旋转期间开启圆形碰撞体，对Overlap敌人施加Impulse向旋转中心牵引 |
| 第4段 | 在空中用手积蓄能量下击 | 圆形范围       | 落地帧触发一次范围圆形碰撞检测，结算伤害后立刻关闭碰撞体     |

**反推总结**：
- 连段之间穿插闪避会从头开始，而完美闪避会从第3段起——反推为普攻GA的 `Retrigger Delay` 设置：完美闪避时GA被中断但保留当前连段索引；普通闪避则触发 `End Ability`，下次激活GA时连段计数器归零。
- 每段普攻为韶光值蓄能——反推为每段动画末尾 `AnimNotify` 触发 `Send Gameplay Event to Actor`（携带Tag `Event.ShaoguangValue.Add`），ASC收到事件后Apply一个叠层GE（`Stacking` 类型，每层加1点韶光值，上限为最大层数）。叠层GE自身有3秒内置CD（携带Tag `Cooldown.Tag.ShaoguangValue`）。

### 三、特殊普攻

触发条件：释放E技能后切入，或变奏入场后直接可用。反推为E技能/变奏入场时给角色挂上 `State.Buff.JinxiEnhanced` Tag——释放特殊技能后，走特殊普攻分支，而非普通普攻分支。

| 段数  | 表现/观察                          | 伤害类型       | 反推实现                                                     |
| :---- | :--------------------------------- | :------------- | :----------------------------------------------------------- |
| 第1段 | 召唤虚影刀下戳                     | 圆形范围       | 在动画第X帧Spawn虚影刀Actor，携带下戳动画+圆形碰撞体，命中后自毁 |
| 第2段 | 召唤四把虚影刀，从四个方向向中冲击 | 圆形范围       | 以目标点或自身为中心，在四个预设偏移位置Spawn四把虚影刀Actor，每把刀携带向心方向Impulse，沿途Overlap造成伤害 |
| 第3段 | 召唤角于前方小范围旋转             | 圆形范围       | 与普通普攻第3段类似，角Actor碰撞体半径缩小，旋转速度加快     |
| 第4段 | 召唤角于前方大范围旋转后向下冲击   | 圆形范围带牵引 | 角Actor先大范围旋转（牵引效果），随后在动画特定帧切换为下冲动画+扩大碰撞体，造成最终段伤害 |

**反推总结**：
- 连段之间穿插闪避或者完美闪避都会保留连段。
- 前3段为韶光值蓄能，反推方式同普通普攻。
- 第4段打完后的“最终技能”——反推为特殊普攻第4段末尾的 `AnimNotify` 触发一个独立GA，该GA播放特写动画+大伤害判定，伤害倍率基于韶光值层数。

### 四、重击/下落攻击

| 触发条件 | 表现/观察                                  | 反推实现                                                     |
| :------- | :----------------------------------------- | :----------------------------------------------------------- |
| 重击     | 先转圈带动小范围转刀，再转圈带动大范围转刀 | 反推为长按普攻键触发重击GA（`Ability Tag` 与普攻不同）。两段伤害对应两个 `AnimNotify`：第一个开启小范围碰撞体，第二个关闭小范围、开启大范围碰撞体。 |
| 下落攻击 | 举刀下劈                                   | 反推为空中按普攻触发下落攻击GA。落地帧Spawn圆形碰撞体，做一次Overlap检测后立刻销毁。 |

**反推要点**：
- 重击消耗体力——反推为Cost GE（扣体力，Duration = Instant）。
- 下落攻击落地范围伤害——反推为落地帧一次性Overlap检测。

### 五、共鸣技能

| 维度         | 分析                                                         |
| :----------- | :----------------------------------------------------------- |
| **观察**     | 向前冲撞后挥刀横砍                                           |
| **冷却时间** | 普通技能3s，特殊技能12s                                      |
| **伤害范围** | 冲撞时为自身圆形，挥刀时为前方扇形                           |
| **特殊机制** | 在平A第3段后释放E技能，会切入特殊空中普攻模组；变奏入场时可直接释放E技能切入特殊普攻 |
| **特殊表现** | 空中特殊普攻打完后，即可释放特写最终技能                     |
| **反推实现** | GA激活后分两阶段：第一阶段 `Add Impulse` 向前冲撞，冲撞期间开启自身圆形碰撞体；第二阶段在冲撞结束帧播放挥刀动画，开启扇形碰撞体。在平A第3段后给角色挂上 `State.Buff.JinxiEnhanced` Tag——后续释放特殊技能后，普攻GA检测到此Tag存在则自动切换到特殊普攻分支。变奏入场时直接挂该Tag，从而实现“直接释放特殊技能”。 |

### 六、共鸣解放

| 维度         | 分析                                                         |
| :----------- | :----------------------------------------------------------- |
| **观察**     | 镜头聚焦到今汐面部，然后瞬间切换至角朝天上飞，吐出激光       |
| **激活条件** | 能量条满才能放（反推为独立FGameplayAttributeData属性“共鸣能量”，通过普攻/技能命中GE回复，大招Cost GE消耗） |
| **伤害机制** | 单段高伤                                                     |
| **反推实现** | 能量满→玩家按键调用激活函数→GA内部 `CommitAbility` 消耗能量+检查CD→`Play Montage And Wait` 播放大招动画（包含特写镜头）→动画过程中 `AnimNotify` 触发镜头切换或定序器片段→伤害帧 `ApplyGameplayEffectToTarget` 结算伤害→`End Ability`。大招GA的 `Activation Owned Tags` 加上 `State.Invincible`，大招期间免疫伤害。 |

### 七、核心被动

| 触发条件                                                     | 效果                 | 反推实现                                                     |
| :----------------------------------------------------------- | :------------------- | :----------------------------------------------------------- |
| 3s为一个周期，自身攻击积攒1点韶光值，队友协同攻击积攒2点韶光值 | 提高最终技能伤害倍率 | 韶光值为独立 `FGameplayAttributeData` 属性。自身攻击→`AnimNotify` 触发GE叠层（1层/3s）；队友协同攻击→通过 `Send Gameplay Event to Actor` 发送信号（Tag `Event.ShaoguangValue.Add`），ASC收到后Apply 2层叠层GE。叠层GE自身带3秒冷却Tag（`Cooldown.Tag.ShaoguangValue`），防止高频攻击无限叠层。最终技能GA激活时读取韶光值属性值，乘以每层倍率系数（从DataTable读取） |

### 八、整体总结

**这个角色的技能组设计逻辑**：
今汐的输出循环为“队友叠协奏→今汐变奏入场→E技能启动特殊普攻模组→打完特殊普攻四段→触发最终技能→大招收尾”。核心数值层数（韶光值）由自身和队友共同积攒，鼓励协奏机制和队友协同攻击。爆发窗口集中在特殊普攻期间，伤害峰值在最终技能和大招。

**如果我需要在 UE5 里复现这个角色**：
- **GA（6个）**：普攻（普通+特殊分支）、重击、下落攻击、E技能、大招、最终技能
- **GE（至少12个）**：普攻伤害GE × 4段、特殊普攻伤害GE × 4段、重击Cost GE、E技能伤害GE、大招伤害GE、韶光值叠层GE、大招无敌GE、能量回复GE
- **GameplayTag（至少8个）**：
  | Tag                           | 类型     | 用途                                                         |
  | ----------------------------- | -------- | ------------------------------------------------------------ |
  | `State.Buff.JinxiEnhanced`    | 状态标记 | 靠 `AnimNotify` 在普攻第3段动画末尾挂给角色，标识可以释放强化技能，以进入强化普攻模组 |
  | `State.Invincible`            | 状态标记 | 大招GA的 `Activation Owned Tags`，免疫伤害                   |
  | `Ability.NormalAttack`        | 技能标识 | 标记普通普攻GA                                               |
  | `Ability.EnhancedAttack`      | 技能标识 | 标记特殊普攻GA                                               |
  | `Ability.E`                   | 技能标识 | 标记共鸣技能GA                                               |
  | `Ability.R`                   | 技能标识 | 标记共鸣解放GA                                               |
  | `Event.ShaoguangValue.Add`    | 事件通知 | `Send Gameplay Event` 发出，通知ASC叠韶光值                  |
  | `Cooldown.Tag.ShaoguangValue` | 冷却管理 | 叠层GE携带，限制3秒内最多叠1次                               |
- **DataTable配置项**：每段伤害数值、韶光值每层倍率系数、韶光值周期（3s）、E技能冷却时间（普通3s/特殊12s）、大招能量消耗、强化状态持续时间
