# 拆解对象：《鸣潮》无冠者

### 一、基础信息

| 维度      | 内容                                                         |
| :-------- | :----------------------------------------------------------- |
| Boss 定位 | 使用长矛打近战，攻击风格偏激进。攻击有时会附带飞天。         |
| 核心机制  | 作为玩家早期遭遇的Boss，大量攻击附带反击圈（弹反提示），用于教学弹反系统。虽仅一个阶段，但连段密集、节奏偏快，通过频繁的弹反窗口引导玩家掌握核心战斗技巧。 |
| 特殊资源  | 无                                                           |

> 玩家需走到场景中间，按下触发按键，无冠者才会从空中俯冲登场。

#### 大招 GA_AerialRampage 详细反推实现

这是无冠者最高威胁的技能，整体为一个多阶段拼接的大招型 GameplayAbility：

1.  **旋转攻击（蓄力）**：GA 激活后，Boss 在原地播放蓄力旋转动画，`AnimNotify` 在碰撞帧开启**矩形碰撞体**，造成第一段伤害（击飞）。
2.  **升空与俯冲**：蓄力后播放起飞动画，角色进入飞行状态。然后依据玩家当前位置，向目标区域进行一次**矩形路径的俯冲命中判定**。
3.  **四段飞天横斩**：俯冲结束后，在空中依次向玩家方向释放四次攻击。每次攻击通过 `AnimNotify` 开启不同朝向的**扇形/矩形碰撞体**，造成高频压制。
4.  **龙卷风（UNIQUE 机制）**：
    -   **行为表现**：Boss 在四段横斩后，向玩家位置召唤一个单次判定的巨大龙卷风。龙卷风**只造成一段伤害**，出现后很快又消失，带有强烈的视觉冲击。
    -   **反推实现**：此段伤害被设计为 `GE_TornadoBurst`（瞬时性的 GameplayEffect），应用**唯一性的 UNIQUE 标签**（如 `Ability.Unique.CrownlessTornado`）。
    -   **机制详解**：在 UE5 中，具有 `UNIQUE` 特性的技能通常通过两种方式实现：一是 GE 自身设置为 `kGameplayEffectApplicationType::Unique`，防止该效果在同一目标上叠加；二是在 Ability 层面设置 `CooldownTags` 或 `BlockAbilitiesWithTag` 来确保该技能在特定时间窗口内仅能触发一次。无冠者的龙卷风很可能同时使用了这两套：**GE 层面确保伤害只生效一次（玩家即使碰触风场边缘也不会再吃第二段伤害），Ability 层面在龙卷风召唤期间阻塞了其他同类技能的释放。**
    -   **碰撞检测**：生成一个**瞬时的球形/圆柱形碰撞体**，命中后 Apply `GE_TornadoBurst`（伤害 + 击飞）。
5.  **收尾打击**：龙卷风结束后，Boss 在空中施展一次**扇形横斩**，然后向斜下方进行**矩形路径的俯冲落地**，造成最后一击。

### 三、受击与韧性反馈

| 受击类型             | Boss 反应        | 反推实现                                                     |
| :------------------- | :--------------- | :----------------------------------------------------------- |
| 普通攻击命中         | 无硬直           | 常驻 `State.Unstoppable` Tag。所有受击 GE 在 `PostGameplayEffectExecute` 中检测到此 Tag 时不触发受击硬直动画，仅扣减韧性值。 |
| 连续攻击命中         | 无硬直           | 同上。                                                       |
| 特殊机制击破（震协） | Boss进入硬直停顿 | 当玩家通过完美闪避/弹反积累震协条满后，按 F 触发震协攻击 → 给自身施加一个短暂的霸体穿透 GE，直接造成 Boss 硬直（硬直 GA）。 |
| 弹反成功             | Boss进入硬直停顿 | 玩家振刀成功时，给 Boss 施加一个带 `State.Parried` Tag 的 GE。行为树检测到此 Tag 时中断当前攻击分支，播放一段固定的硬直/踉跄动画，并在此期间 Boss 受到的伤害提升（如有）。硬直时长和易伤系数从 DataTable 读取。 |
| 韧性条被击破         | Boss 倒地瘫痪    | 常规韧性条（架势条）归零 → 触发瘫痪 GA，Boss 强制倒地一段时间。期间可被玩家输出。起身时强制调用 GA_BreakRecover（两段横斩）。 |

**反推总结**：
- 无冠者全程霸体，常规攻击无法造成硬直。这迫使玩家必须学习弹反或利用震协来创造输出窗口，符合教学目的。
- 硬直触发完全依赖特定 GameplayEvent（弹反成功、震协触发、架势击破），与普通伤害量无关。
- 韧性条被击破后的起身反击是固定行为，属于 AI 行为树分支，避免玩家简单“打桩”。

### 四、整体总结

**这个 Boss 的设计逻辑**：

无冠者是《鸣潮》战斗系统的“新手老师”。它的所有攻击几乎都附带明显的反击圈提示，鼓励玩家使用振刀（弹反）。技能连段密度高、压迫感强，但通过弹反可频繁打断其节奏，给予玩家强烈的正反馈。这是典型的“风险-回报”教学：你不弹反就会被持续压制，敢弹反就能反客为主。另外，无冠者不存在复杂的阶段转换，保证玩家在最初几个小时内的认知负荷可控。

**如果我需要在 UE5 里复现这个 Boss**：

- **GA（约 8～10 个）**：
  - GA_Intro（入场俯冲+横扫）
  - GA_ComboA（三连：横斩/直砸/上挑）
  - GA_ComboRush（冲刺五连）
  - GA_ChargeAssault（前冲二连+砸地）
  - GA_SimpleCombo（二连）
  - GA_TripleSlash（三横斩）
  - GA_AerialRampage（飞天大招，含龙卷风）
  - GA_BreakRecover（起身二连）
  - GA_Phase2Unlock（低空冲，条件性）

- **GE（约 10+）**：
  - GE_Damage_Standard（普通伤害 GE，可配系数）
  - GE_Knockback（击飞）
  - GE_Parried (State.Parried Tag，触发硬直)
  - GE_Unstoppable_Passive（常驻，赋予 State.Unstoppable）
  - GE_TornadoBurst（龙卷风伤害 GE，Unique 应用，单次伤害）
  - GE_DamageVulnerable（弹反成功后 Boss 进入的易伤状态，可选）

- **GameplayTag（至少 5 个）**：
  - `State.Unstoppable`（霸体）
  - `State.Parried`（被弹反硬直）
  - `State.StaggerBroken`（韧性归零倒地）
  - `Ability.Boss.Crownless`（所有无冠者 GA 的通用 Tag，方便管理）
  - `Ability.Unique.CrownlessTornado`（龙卷风唯一性 Tag）
  - `Event.StaggerBreak`（架势击破事件）

- **行为树分支（约 3～4 个核心分支）**：
  - **Idle/Patrol**：待机，监测玩家距离和触发条件。
  - **Combat**：主战斗逻辑。根据与玩家距离、玩家行为（是否频繁弹反）动态选择 GA（近身多用 GA_SimpleCombo/GA_TripleSlash，中距离用 GA_ComboRush，低血量或满足条件时释放 GA_AerialRampage）。
  - **Staggered/Parried**：检测到 State.Parried 或 State.StaggerBroken 时，中断 Combat，播放硬直动画；StaggerBroken 结束后强制进入 GA_BreakRecover。
  - **Phase2 Unlock**：当黑板 `bPhase2Unlocked == true`，Combat 中额外加入 GA_Phase2Unlock 的选择。

- **DataTable 配置项（示例）**：
  - 各 GA 的基础伤害系数、击飞距离
  - 横扫/直砸的扇形角度、矩形长宽
  - 韧性条最大值、每次受击扣减值
  - 弹反硬直持续时间
  - 倒地瘫痪时间
  - 龙卷风瞬时碰撞体半径、伤害系数
  - 低空冲刺速度、路径贴花长度
  - AI 侦测距离阈值（脱离战斗距离）
